# 基于大模型驱动的文章自动评分方法研究 (Data Repository)

本仓库包含了论文《基于大模型驱动的文章自动评分方法研究》中所使用的核心数据集。本项目旨在提出一种多维语义解构与对比校准的协同评分框架，以期在实际评估环境（actual evaluation environments）和线上推理阶段（online inference stages）提供稳定、一致且可解释的自动化文本质量度量。

## 📁 目录结构与数据说明

本项目的数据集按照论文的章节逻辑与实验递进关系分为四大类，统一放置在 `dataset/` 目录下，均采用标准的 `.jsonl` 格式存储。

### 1. 基础多维评分数据集 (Base Absolute Scoring)
* **文件路径**: `dataset/01_aes_absolute_scoring_base.jsonl`
* **数据规模**: 8,000 篇中文文章。
* **数据划分**: 训练集 (6,000 篇) / 独立测试集 (2,000 篇)。
* **标注范式**: 采用 70% 大语言模型（基于 CoT-PE 思维链）与 30% 专家独立盲审校验的混合标准。
* **用途**: 用于论文第三章，训练大语言模型主干网络与多层感知机（MLP）的特征映射与加权融合。

### 2. 相对偏好对比数据集 (Pair-wise Preference)
* **文件路径**: `dataset/02_aes_pairwise_preference.jsonl`
* **数据规模**: 16,000 条数据（由 8,000 组原始对比对构成）。
* **数据特征**: 包含严格的对称翻转增强（Symmetric Flipping Augmentation），正负样本比例严格控制在 1:1，以消除位置偏置（Position Bias）。
* **用途**: 用于论文第四章，训练深度判别对比网络，优化监督排序与循环一致性损失。

### 3. 跨文体泛化测试集 (OOD-Set)
* **文件路径**: `dataset/03_aes_ood_generalization_test.jsonl`
* **数据规模**: 200 篇样本（100 组对比对）。
* **数据分布**: 涵盖科技说明文与文学散文，与基础训练集分布显著不同。
* **用途**: 用于论文第四章，评估模型在未见学科语境下的泛化能力与特征解构的鲁棒性。

### 4. 优秀文章离线锚点库 (Offline Benchmark Anchors)
* **文件路径**: `dataset/04_aes_offline_benchmark_anchors.jsonl`
* **数据规模**: 500 篇精选样本。
* **数据特征**: 分数处于全局分布的 Top 10% 且打分方差极小的高质量物理基准。
* **用途**: 用于论文第四章的基准校准（Calibration）阶段，通过线性插值将相对偏好概率映射为高精度的绝对分数。
